# 🖼️ Image Filter Tool

A simple Python project to apply filters to images using PIL and NumPy.

---

## 🚀 Features
- Grayscale
- Sepia
- Invert Colors
- Darken
- Brightness Adjustment
- Contrast Control
- Black & White (Thresholding)
